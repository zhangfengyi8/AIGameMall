# AIGameMall

游戏账号 C2C 交易 AI 智能导购项目。

本仓库采用 monorepo 架构，当前阶段聚焦 Python 后端、Agent 编排与 JSON 数据查询能力。

## 目录概览

- `apps/web`: 用户侧纯 HTML/JS 前端。
- `services/api`: Python 业务 API 服务。
- `services/agent-orchestrator`: Python Agent 编排服务。
- `packages`: 后端共享 Python 包。
- `data`: JSON 数据目录，用于账号、标签与样例数据查询。
- `docs`: 产品、架构、API 与 Agent 设计文档。
- `configs`: 仓库级配置。
- `scripts`: 本地开发辅助脚本。
